# poli-page (Python SDK)

Official Python SDK for [Poli Page](https://poli.page) — generate PDFs from any Python application, framework or script.

> **Status**: scaffold only. Implementation begins in P1.0 of the [SDK roadmap](https://github.com/poli-page/poli-page/blob/develop/docs/onboarding/micka/sdk-roadmap.md).

## Install

```bash
pip install poli-page
```

## Quick start

To be filled in as the SDK is built. The client will expose both a synchronous and an asynchronous variant, mirroring the public API contract defined in `docs/onboarding/micka/sdk-specification.md` of the platform repo.

## Recipes

Framework-agnostic recipes (Flask, plain script, Celery, Jupyter) live under `examples/` once implemented. First-party integrations for Django and FastAPI live in their own repositories: [`poli-page/django`](https://github.com/poli-page/django) and [`poli-page/fastapi`](https://github.com/poli-page/fastapi).

## Publishing

Published to **PyPI** as [`poli-page`](https://pypi.org/project/poli-page/).

Full Poli Page documentation is at [docs.poli.page](https://docs.poli.page).
